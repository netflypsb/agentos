# AgentOS Platform Launch Plans

## Executive Summary

AgentOS will launch on **two complementary platforms** to maximize revenue speed and market coverage:
1. **MCP Marketplace** - Primary revenue channel with subscription model
2. **Apify Actors** - Secondary revenue channel with pay-per-event model

This dual-platform approach targets different user segments and maximizes total addressable market.

---

| Platform | Revenue Model | Time to Revenue | User Type | Strategic Value |
|---|---|---|---|---|
| **MCP Marketplace** | Subscription ($15/mo) | 3-7 days | MCP-native developers | Primary revenue, high LTV |
| **Apify Actors** | Pay-per-event ($0.01-0.05/call) | 5-10 days | No-code/low-code users | Volume users, market expansion |

#### Day 9-10: Code Conversion
**Time Investment: 12 hours**

```typescript
// Convert existing MCP server to Apify Actor format:

// src/actor.ts (new file)
import { Actor } from 'apify';
import { McpServer } from '@modelcontextprotocol/sdk/server/mcp.js';
import { createServer } from './server.js';

await Actor.init();

// Expose MCP server via HTTP for Apify
const server = await createServer();
const port = process.env.APIFY_ACTOR_STANDBY_PORT || 8080;

// Create HTTP server to handle MCP requests
const httpServer = http.createServer(async (req, res) => {
  // Handle MCP protocol over HTTP
  await handleMcpRequest(req, res, server);
});

httpServer.listen(port, () => {
  console.log(`AgentOS MCP Actor listening on port ${port}`);
});

await Actor.exit();
```

#### Day 11: Pay-Per-Event Pricing Setup
**Time Investment: 4 hours**

```typescript
// Add Apify charging to each tool call
export async function withApifyCharge(toolName: string, handler: ToolHandler) {
  return async (args: ToolArgs) => {
    // Charge for tool usage
    const chargeResult = await Actor.charge({
      eventName: `${toolName}_call`,
      amount: getToolPricing(toolName) // $0.01-0.05 per call
    });
    
    if (chargeResult.eventChargeLimitReached) {
      throw new Error('Daily spending limit reached');
    }
    
    return await handler(args);
  };
}

// Tool pricing configuration
const TOOL_PRICING = {
  'format_convert': 0.01,    // $0.01 per conversion
  'kv_set': 0.005,          // $0.005 per store
  'kv_get': 0.001,          // $0.001 per retrieve
  'task_create': 0.02,      // $0.02 per task
  'budget_check': 0.005,    // $0.005 per check
  'template_render': 0.015  // $0.015 per render
};
```

## Risk Mitigation

### Platform-Specific Risks
| Risk | MCP Marketplace | Apify Actors | Mitigation |
|---|---|---|---|
| **Approval Delays** | 24-48hr review | 12-24hr review | Submit early, have backup plan |
| **Technical Issues** | License validation | HTTP transport | Thorough testing, fallbacks |
| **Low Adoption** | Niche audience | Competition | Cross-promotion, free tier |
| **Revenue Issues** | High churn | Low volume | Diversified pricing models |

### Mitigation Strategies
1. **Staggered Launch**: MCP first, Apify second
2. **Free Tier Always Available**: No platform lock-in
3. **Cross-Platform Compatibility**: Same core codebase
4. **Multiple Revenue Streams**: Subscriptions + per-event
